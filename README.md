# LogLens
Log Analysis and Alerting Tool 

Features:
1. Regex-based log parsing for structured & unstructured formats
2. Real-time log categorization and summarization
3. Automated email alerts for error-level logs via Gmail SMTP

Update Credetials before running the application
1. create app pass in gmail
2. update application.properties
3. update email address in code to whom you want to trigger the email.

Steps to run:
1. git clone https://github.com/amarkrsoni/loglens.git
2. cd loglens
3. mvn clean install
4. mvn spring-boot:run
